Shiki Copy Button Transformer is a Shiki based transformer that adds a copy button on a code block. It is highly customizable and supports dark mode.

## Features

1. Dark mode support 🌑

2. Highly customisable 🎨

3. Duration 🕑

4. Display mode 👀
